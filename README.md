# One-way anonymous chat with encryption

---
Install Termux:
`pkg install php`
`pkg install git`
`git clone https://github.com/master-off/chat.git`
`cd chat`

---
Server:
`php server.php`
`Далее пишем желаемый порт и ключ шифрования`

---
Client:
`Вписываем тот же порт и ключ котрый указал сервер`
